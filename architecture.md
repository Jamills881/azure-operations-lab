# Architecture

## Phase 1

The initial Azure environment will contain:

### Resource Group

Purpose:
Provide a logical container for all project resources.

### Storage Account

Purpose:
Store project files and provide exposure to Azure Storage concepts.

### App Service

Purpose:
Host a simple web application that can later be monitored, secured, and automated.
